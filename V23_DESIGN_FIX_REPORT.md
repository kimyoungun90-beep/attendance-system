# v23 엑셀 디자인 수정 내역

## 요청 반영

1. `계획&근태 상이 인원`의 `구분 색상 안내`
   - 기존: 한 개 셀에 4개 문구를 넣어 Excel에서 글씨가 모두 검은색으로 표시됨
   - 수정: 4개 안내 항목을 각각 별도 셀로 분리하고 빨강·주황·파랑·초록 글씨와 연한 배경색 적용

2. `출근 미등록` 시트
   - 네이비 제목 영역과 기준일 안내
   - 상단 요약 카드 5개: 총 미등록 건수 / 대상 인원 / 대상 매장 / 미처리 / 처리 완료
   - 서울·경인·충청·경북·경남·전라 지역별 그룹
   - 지역 구분 행에 지역별 건수, 증빙 O, 미처리, 완료 집계
   - 헤더 행 고정, 눈금선 숨김
   - 빨간 오류 배지, 노란 K열 증빙 입력 칸, 처리상태 표시
   - 증빙 입력 위치는 기존 요구사항과 동일하게 K열 유지

## 유지 사항

- 증빙 O 인식 및 재분석 처리 로직 유지
- 발생일 대체휴무 계산 로직 유지
- D1 데이터 및 마이그레이션 변경 없음
- 같은 경로·같은 월 재등록 시 기존 월 자료 교체 방식 유지
