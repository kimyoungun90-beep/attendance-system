# V51 엑셀 생성 오류 수정 보고

## 원인
V50에서 출근증빙·휴무확인 수기 입력 영역을 추가하는 과정에서 수기 입력용 변수(`manualRows`, `manualTitleRow`)가 의도한 `출근증빙·휴무확인` 시트가 아니라 일부 다른 시트 생성 로직에 섞여 들어갔습니다. 이로 인해 엑셀 생성 중 정의되지 않은 변수를 참조해 다운로드가 중단될 수 있었습니다.

## 수정
- 수기 입력 영역은 `출근증빙·휴무확인` 시트에서만 생성되도록 정리
- 다른 시트에 잘못 들어간 수기 입력 영역 관련 로직 제거
- `출근증빙·휴무확인` 시트 지역별 → 이름순 → 매장 → 날짜순 정렬 유지
- 수기 입력 이름 → 사번 자동 표시 수식 유지
- 수기 입력 출근확인(O) 재업로드 시 09:00 반영 유지
- V50의 app.js 수기 증빙 읽기 로직 유지
- 상단 버전 v51 표시
