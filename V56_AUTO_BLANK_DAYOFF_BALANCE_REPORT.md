# V56 자동 휴무 공백 보정 및 초과 휴무 증빙 대상 조정

- 상담사근태 생성 전, 기본 휴무 기준 안에서만 공백 미출근일을 `휴무(공백)`으로 자동 표시합니다.
- 전자랜드처럼 월 기준 휴무가 8개인 경우 8개까지만 자동 보정하며, 기준을 초과해서 자동 휴무를 만들지 않습니다.
- 등록/사용된 휴무가 기준을 초과하면 대체휴무+보상휴가 잔여가 있는 초과일은 `휴무(잔여)` 살구색으로 표시합니다.
- 잔여가 없는 초과일은 `휴무(초과)` 빨간색으로 표시합니다.
- `휴무(공백)`은 출근증빙·휴무확인 시트에서 제외하고, 대체/보상/연차 처리가 필요한 `휴무(잔여)`, `휴무(초과)`만 증빙 대상에 추가합니다.
- 증빙 완료 파일로 대체/보상/연차 등을 확정하면 상담사근태가 해당 확정 상태로 바뀌어 일반 색상으로 돌아갑니다.
- D1 구조 변경은 없습니다.
