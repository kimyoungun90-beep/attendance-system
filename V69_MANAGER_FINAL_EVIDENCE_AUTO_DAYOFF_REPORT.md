# v69 관리자반영 기준 증빙·자동휴무 정리

## 반영 사항

1. `출근 미입력`, `출ㆍ계 미입력`, `근태 미입력`, `미등록` 등 미출근/미계획 표기를 기본 휴무 자동 보정 대상으로 확장했습니다.
2. 기본 휴무 기준(전자랜드 6월 8개 등)을 못 채운 상담사는 증빙으로 띄우기 전에 해당 미입력 날짜를 `휴무(공백)`으로 자동 처리합니다.
3. 위 처리는 `상담사근태`와 `상담사근태_관리자반영` 양쪽에 적용됩니다.
4. 매니저별 월마감 수정본 값은 `상담사근태_관리자반영`에 반영합니다.
5. 매니저 수정본에 수기로 입력한 `출근`, `근무`, `09:00` 같은 출근성 값은 실제 출근증빙 확인 전에는 반영하지 않습니다.
6. `출근증빙·휴무확인`은 관리자반영 기준으로 남은 미입력/미등록 건만 표시하도록 보강했습니다.
7. 불필요한 `상담사근태_증빙반영`, `관리자수정 충돌` 시트는 유지하지 않습니다.
