# v27 수정 내역

## 1. 계획 공백 + 실제 출근

근무계획이 공백이어도 실제 출근시간, 변경 출근시간 또는 실제근태가 출근·근무·정상으로 확인되면 최종 표시를 `출근`으로 확정합니다.

해당 건은 다음 이상 목록에서 제외됩니다.

- 출근 미등록
- 계획&근태 상이 인원
- 전체 요약본
- 매니저별 이상 근태

상담사근태의 `출근 등록 횟수`와 `총 등록 현황`에는 정상 출근으로 포함됩니다. 계획과 출근이 모두 없는 날짜만 `출ㆍ계 미입력`으로 남으며 등록 횟수에서는 제외됩니다.

## 2. 휴가 색상

- 대체휴무·대체휴일 정상 사용: `#A9D08E`
- 보상휴가 정상 사용: `#548235` + 흰색 글씨
- 대체휴무 초과 사용일: 빨간색
- 보상휴가 초과 사용일: 빨간색

초과 색상은 전체 사용일을 빨갛게 바꾸지 않고, 실제 잔여 부족이 발생한 날짜에만 적용합니다.
