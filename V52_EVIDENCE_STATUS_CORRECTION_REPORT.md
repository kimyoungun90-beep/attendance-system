# V52 출근증빙 상태 보정 확장 보고

## 반영 내용
- `출근증빙·휴무확인` 시트를 단순 출근/휴무 확인에서 전체 상태 보정 시트로 확장했습니다.
- 자동 목록과 수기 입력 영역 모두 동일한 처리 항목을 사용합니다.
- 처리 항목: 출근확인, 휴무확인, 연차확인, 오전반차, 오후반차, 출산휴가, 육아휴직, 공가, 경조, 대체휴무 1일, 대체휴무 0.5일, 보상휴가 1일, 보상휴가 0.5일.
- 수기 입력 영역에서 이름과 발생일을 입력하고 해당 처리 칸에 O를 넣으면 재업로드 시 상담사근태 해당 날짜에 반영됩니다.
- 상담사근태 날짜 셀도 출근증빙·휴무확인 시트의 K~W열 보정값을 참조하도록 확장했습니다.
- 연차→휴무, 휴무→연차, 연차→출근, 공백→출근 등 수기 보정 흐름을 지원합니다.
