## 주차별 확인 수정
- 주 6회 이상: 실제 출근기록 6회 이상이면 표시.
- 주 6회 후보: 실제 출근 + 공백/미출근 확인 대상 합계가 6회 이상이 될 수 있으면 표시.
- 주 3회 이하 공백: 실제 출근 3회 이하이면서 공백/미출근 확인 대상이 남아 있으면 표시.
- 출근 4회 + 휴무 2회 + 공백 1회는 출근 가능성이 5회이므로 제외.
- 출근 5회 + 휴무 1회 + 공백 1회는 공백이 출근이면 6회가 될 수 있으므로 표시.
- 출근 4회 + 휴무 1회 + 공백 2회는 공백 2회가 모두 출근이면 6회가 될 수 있으므로 표시.
- 출근 3회 + 휴무 2회 + 대체 1회 + 연차 1회처럼 7일이 모두 설명되면 제외.
