# V59 자동 휴무(공백) 및 공가·경조 색상 수정

## 수정 내용

1. 상담사근태에서 계획/근태가 모두 비어 있는 공백일은 기본 휴무 기준 안에서 `휴무(공백)`으로 자동 보정되도록 보강했습니다.
   - 기존 표시가 `출ㆍ계 미입력`, `미입력`, `미등록`으로 만들어진 경우도 공백 미출근일로 인식합니다.
   - 자동 보정은 기본 휴무 기준까지만 적용하며 초과분은 자동 휴무로 만들지 않습니다.

2. 공가·경조는 회사 인정 사유로 처리하도록 수정했습니다.
   - 연차신청현황 승인 여부와 무관하게 빨간색 경고를 적용하지 않습니다.
   - 상담사근태에서는 기존 인정 색상으로 표시합니다.
   - 출근증빙·휴무확인 시트의 휴가 승인확인 대상에서도 제외됩니다.

3. `휴무(공백)`은 기존 휴무 색상을 유지하고 증빙 시트에 표시하지 않습니다.

4. 휴무 기준 초과 건만 대체+보상 대체 인원 또는 후속 확인 대상으로 이어집니다.
