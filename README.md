# 전자랜드 · 홈플러스 근태 관리 v3

기존 운영 주소와 Cloudflare D1을 그대로 사용하면서 최종 확정 요구사항을 반영한 버전입니다.

- GitHub 저장소: `kimyoungun90-beep/attendance-system`
- 운영 주소: `https://attendance-system-wzz.pages.dev/`
- Cloudflare D1: `attendance-manager-db`

## v3 핵심 변경

1. 사용자 화면의 구분 명칭을 전부 **경로**로 통일
2. 개인별 대체휴무 부여 기능 완전 제거
3. 경로·발생 월별 전 직원 공통 대체휴무 설정 추가
4. 같은 경로·같은 월 마감은 추가 저장하지 않고 기존 월 마감을 완전히 교체
5. 과거 월 교체 또는 공통 부여 설정 수정·삭제 시 이후 월 자동 재계산
   - 직원별 연차 누적
   - 대체휴무 차감
   - 다음 달 이월
   - 사용 종료일 만료
   - 대체휴무 초과 사용
6. 엑셀 다운로드 시 아래 7개 결과 시트 생성
   - `1_근무인데 출근없음`
   - `2_휴무휴가인데 출근`
   - `3_계획실제 불일치`
   - `4_기본휴무 초과`
   - `5_대체휴무 잔여`
   - `6_대체휴무 초과`
   - `7_연차 등록누적`

## 자동 판정 기준

### 출근기록이 필요한 계획

- 근무
- 근무A
- 근무B
- 근무C
- 교육
- 오전반차
- 오후반차
- 공백

실제 출근시간과 변경 출근시간이 모두 없으면 `근무인데 출근기록 없음`으로 표시합니다.

### 출근기록이 있으면 이상으로 보는 계획

- 휴무
- 무급휴가
- 연차
- 공가
- 휴가
- 경조
- 대체휴일(1일)
- 대체휴일(0.5일)
- 보상휴가(1일)
- 보상휴가(0.5일)

위 계획인데 출근기록이 있으면 `휴무·휴가인데 출근기록 있음`으로 표시합니다.

`기타`는 자동 단정하지 않고 `계획·실제 근태 불일치` 검토 대상에 표시합니다.

## 기본 휴무 및 누적 계산

- 홈플러스: 월 6일
- 전자랜드: 해당 월 토요일+일요일 개수
- 기본 휴무 계산에는 계획표의 `휴무`만 포함
- 대체휴일·보상휴가와 기본 휴무 초과분은 대체휴무 사용으로 계산
- 연차 1일, 오전반차 0.5일, 오후반차 0.5일로 누적

## 현재 v1 운영 사이트 업데이트 순서

### 1. D1 마이그레이션 실행

Cloudflare 대시보드에서 다음 순서로 이동합니다.

1. **Workers & Pages**
2. **D1 SQL Database**
3. `attendance-manager-db`
4. **Console**
5. `migrations/0002_route_monthly_management.sql` 전체 내용을 붙여넣기
6. **Execute** 클릭

이 SQL은 현재 v1 DB에서 **한 번만 실행**합니다.

### 2. GitHub 파일 교체

ZIP 안의 아래 파일과 폴더를 저장소 루트에 그대로 덮어씁니다.

- `public/index.html`
- `public/assets/app.js`
- `public/assets/styles.css`
- `functions/_lib/recalculate.js`
- `functions/api/closures.js`
- `functions/api/closures/[id].js`
- `functions/api/substitute-grants.js`
- `functions/api/substitute-grants/[id].js`
- `functions/api/substitute-balances.js`
- `migrations/0002_route_monthly_management.sql`

기존 v2 초안의 개인별 대체휴무 테이블과 화면은 사용하지 않습니다.

### 3. 배포 확인

GitHub 커밋 후 Cloudflare Pages 배포가 완료되면 아래 항목을 확인합니다.

1. 운영 주소 접속
2. 상단에 `D1 영구 저장 연결` 표시
3. 경로별 공통 대체휴무 메뉴 정상 표시
4. 홈플러스 또는 전자랜드 파일 분석
5. 월 마감 교체 저장
6. 같은 경로·같은 월을 다시 저장했을 때 월 마감 기록이 1건만 유지되는지 확인
7. 7개 시트 엑셀 다운로드 확인

## 재계산 예시

- 홈플러스 2026-06 공통 부여 4일
- 사용기간 2026-06-01 ~ 2026-07-31
- 직원 A가 6월 1일 사용
- 7월 시작 잔여 3일
- 7월 4일 사용
- 7월 결과 `대체휴무 1일 초과 사용`
- 7월 31일까지 남은 미사용분은 8월로 이월하지 않고 만료

과거 6월 마감 파일을 교체하면 6월 이후 저장된 홈플러스 월 마감을 모두 다시 계산합니다.
