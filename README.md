## v67 출/계 미입력 자동 휴무 보강

- 출/계 미입력, 출ㆍ계 미입력, 미입력, 미등록 표기를 모두 공백 미출근일로 인식합니다.
- 전자랜드 기본 휴무 8개 등 월 기준 휴무를 덜 쓴 상담사는 출/계 미입력일을 휴무(공백)으로 자동 보정합니다.
- 상담사근태와 상담사근태_관리자반영 두 시트 모두 동일하게 자동 보정합니다.
- 관리자반영 후 휴무 개수가 다시 8개 미만이 되는 경우에도 상담사근태_관리자반영에서 남은 출/계 미입력일을 휴무(공백)으로 채웁니다.
- 출근증빙·휴무확인 시트에서도 자동 휴무(공백) 처리된 건은 증빙 대상으로 띄우지 않습니다.


## v68 실제 출근자 경고색 제외
- 실제 출근시간/변경 출근시간/출근 증빙이 있는 날짜는 계획이 휴무·연차·대체·보상 등으로 되어 있어도 정상 출근으로 봅니다.
- 출근을 찍은 날짜는 상담사근태와 상담사근태_관리자반영에서 빨간색 경고색, 휴무 초과 빨간색, 계획·근태 상이 경고 대상에서 제외합니다.
- 출근하지 않은 미입력·휴무초과·대체/보상 초과 건만 기존처럼 확인 대상으로 표시합니다.

- 자동 휴무(공백)는 상담사근태와 상담사근태_관리자반영 두 시트 모두에 반영됩니다.

- 출근증빙·휴무확인 시트는 관리자반영 기준으로 남은 미입력/미등록 건만 표시하도록 보강했습니다.
