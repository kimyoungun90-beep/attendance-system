# 근태 관리 시스템 v31

## v31 핵심

1. `이전 최종본 휴가 사용내역 저장` 화면을 별도로 두지 않고 `월별 파일 통합 관리` 화면으로 합쳤습니다.
2. 홈플러스와 전자랜드 파일 선택 영역을 분리하여 같은 대상 월의 두 경로 파일을 한 화면에서 각각 등록할 수 있습니다.
3. 사용자가 `파일 구분`을 직접 고르지 않습니다. 파일명과 시트명을 기준으로 계획표·실제 근태표·최종본·기타를 자동 분류합니다.
4. 상담사근태 시트가 있는 최종본은 파일 보관과 동시에 대체휴무·보상휴가 사용내역을 D1 누적 장부에 자동 반영합니다.
5. 저장 목록에 최종본 휴가내역 `반영 완료/미반영` 상태를 표시하고, 필요하면 `휴가내역 반영` 또는 `휴가 재반영` 버튼으로 다시 처리할 수 있습니다.
6. 같은 경로·같은 월·같은 자동 분류 파일은 기존 보관 파일을 교체합니다.
7. 기존 월 마감, 대체휴무·보상휴가 계산, 증빙 O 처리, D1 누적자료는 삭제하지 않습니다.
