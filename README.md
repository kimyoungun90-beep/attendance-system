## v43 추가

- 상담사근태의 정상 출근 표시는 원본 `(실제)출근시간` 값(예: `09:03`)으로 출력하고 음영을 제거했습니다.
- 계획이 근무 외 코드인데 실제 출근한 날짜는 출근시간과 노란색을 표시하며, 교육은 정상 출근으로 처리합니다.
- 연차·반차, 휴무, 대체휴무·보상휴가, 초과 사용, 양쪽 미입력 색상을 새 기준으로 조정했습니다.
- 수정 최종본의 날짜별 시간·연차·반차·휴무·대체휴무·보상휴가·빈칸 변경을 해당 월 최종 확정값으로 다시 반영합니다.
- 출근 미등록, 계획&근태 상이, 휴무 초과자의 처리 완료를 직원번호·날짜·이상 유형 또는 직원번호·회사·기준월로 연결합니다.
- 수정 최종본 재등록 후 상담사근태, 전체 요약본, 매니저별 이상 근태, 주 근태 확인자, 이상자 시트, 휴가 사용량·잔여를 전체 재계산합니다.
- 근무계획·근태 원본은 열 위치가 아니라 헤더명과 실제 날짜 헤더를 기준으로 인식합니다.
- 상담사근태 창 고정은 상단 6행, 왼쪽 M열까지 그대로 유지합니다.
- 기존 D1 구조를 그대로 사용하며 별도 마이그레이션은 필요하지 않습니다.

상세 내용은 `V43_ACTUAL_CLOCK_RECALCULATION_REPORT.md`를 확인하세요.

## v35 핵심 변경

- 기존 D1 구조와 월 마감 저장 방식은 변경하지 않았습니다. 별도 D1 마이그레이션은 필요하지 않습니다.
