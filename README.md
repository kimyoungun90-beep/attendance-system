# 근태 관리 시스템 v30

## v30 핵심

1. 이전에 만들어 둔 최종본의 **상담사근태 시트만** 읽어 대체휴무·보상휴가 사용내역을 D1에 저장할 수 있습니다.
2. 요약 수식, 휴무 개수 수식, 다른 시트와 수식 문자열은 읽지 않습니다.
3. 상담사근태의 사번·입사일·매장정보도 월별 인력 스냅샷에 비파괴 방식으로 보완합니다.
4. 경로 일괄 보상휴가는 발생일 실제 출근자만 입력한 일수만큼 부여합니다.
5. 발생일 미출근자는 보상휴가를 부여하지 않고 해당 월 기본휴무를 1일 추가합니다.
6. 대체휴무는 v29와 동일하게 출근·휴무 여부와 관계없이 부여하며 기본휴무를 가감하지 않습니다.
7. 기존 대시보드 디자인, 7행 고정, 전체 시트 70% 배율, 증빙 O 반영, 같은 경로·같은 월 교체 저장은 유지합니다.
