## v35 핵심 변경

- `계획&근태 상이 인원`에서 처리여부에 O를 입력하면 처리상태 셀과 입력 셀이 Excel에서 즉시 연녹색으로 변경됩니다.
- `계획&근태 상이 인원`과 `휴무 초과자`의 우측 상단 구분 색상 안내를 실제 색상 블록으로 복원했습니다.
- `휴무 초과자`에 전월까지 사용 후 이월된 `대체휴무 잔여`, `보상휴가 잔여`, `이월 합계`를 추가했습니다.
- 휴무 초과자 처리 입력 위치는 새 열 구성에 따라 T열, 처리상태는 U열입니다.
- 비교 기준일 이후 날짜는 비교 최종본 차이 및 전체 요약본에 표시하지 않습니다.
- 계획&근태 상이 시트에서 처리 완료한 사번·일자는 증빙 최종본 재업로드 후 전체 요약본과 최종본 비교 차이에서 제외됩니다.
- 기존 D1 구조와 월 마감 저장 방식은 변경하지 않았습니다. 별도 D1 마이그레이션은 필요하지 않습니다.

상세 내용은 `V35_PROCESSING_CARRYOVER_CUTOFF_REPORT.md`를 확인하세요.
