# V57 Dayoff Replacement Separate Sheet

## 변경 요약
- 휴무 초과자 중 대체휴무·보상휴가로 처리 가능한 건을 `출근증빙·휴무확인`에서 제외했습니다.
- 3번째 시트 `대체+보상 대체 인원`을 추가했습니다.
- 해당 시트에서 대체/보상 처리 가능 건과 불가능 건을 구분합니다.
- 해당 시트의 O 입력도 증빙 완료 파일 재업로드 시 상담사근태와 잔여 재계산에 반영되도록 파싱 대상에 추가했습니다.
- 기존 D1 구조 변경은 없습니다.

## 시트 순서
1. 상담사근태
2. 출근증빙·휴무확인
3. 대체+보상 대체 인원
4. 계획&근태 상이 인원
5. 휴무 초과자

## 처리 기준
- 가능: 기본 휴무 초과분을 대체휴무/보상휴가 잔여로 처리 가능
- 불가능: 대체+보상 잔여로 부족하여 연차 확인 필요
- 증빙 완료 파일 재업로드 시 O 입력값을 기준으로 재계산
