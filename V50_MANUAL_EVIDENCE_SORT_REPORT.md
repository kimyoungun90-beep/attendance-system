# V50 수기 출근증빙·지역별 이름순 정렬 수정 보고

## 반영 내용
- `출근증빙·휴무확인` 시트 정렬을 지역별 이름순 기준으로 변경했습니다.
- 해당 시트 하단에 `수기 입력 영역`을 추가했습니다.
- 계획이 휴무/공백 등이라 자동 목록에 뜨지 않았지만 출근 증빙을 받은 인원은 수기 입력 영역에 이름과 발생일을 입력하고 `출근확인(O 입력)`에 O를 넣어 처리할 수 있습니다.
- 수기 입력 영역의 사번 칸은 `상담사근태` 시트의 이름을 기준으로 자동 표시되도록 수식을 넣었습니다.
- 재업로드 시 사번 수식값이 비어 있더라도 이름+매장 기준으로 상담사근태에서 사번을 찾아 출근증빙을 반영하도록 보강했습니다.
- `출근확인` 처리 시 상담사근태 해당 날짜는 기존 기준대로 `09:00`으로 반영됩니다.
- `휴무확인` 처리 시 상담사근태 해당 날짜는 `휴무`로 반영됩니다.

## 주의
- 동명이인이 있는 경우에는 자동 사번이 첫 번째 이름으로 잡힐 수 있으므로 사번을 직접 입력하는 것이 안전합니다.
- D1 구조 변경은 없습니다.
