# v28 수정 내역

## 디자인 개편
- `휴무 초과자`, `전체 요약본`, `매니저별 이상 근태`를 기존 `계획&근태 상이 인원`, `출근 미등록`과 같은 네이비 대시보드형으로 변경했습니다.
- 각 시트에 제목, 설명, 요약 카드 5개, 색상 안내, 네이비 헤더, 서울·경인·충청·경북·경남·전라 지역 구분 행을 적용했습니다.
- 지역별 구분 행에는 해당 지역의 유형별 건수를 표시합니다.

## 고정 및 실행 보기
- 다섯 개 대시보드 시트는 7행까지 고정되어 8행부터 스크롤됩니다.
- 생성되는 모든 시트의 눈금선을 숨기고 기본 확대 비율을 70%로 저장합니다.
- XLSX 생성 후 worksheet XML의 `sheetView`와 `pane`을 다시 검증하여 Excel 실행 시 70% 및 7행 고정이 실제로 적용되도록 보강했습니다.

## 데이터 보존
- 기존 근태 판정, 증빙 O 처리, D1 월 교체 저장, 대체휴무·보상휴가 계산 로직은 변경하지 않았습니다.
- 시트 표시와 엑셀 보기 설정만 수정했습니다.
