# V63 관리자 월마감 수정본 반영 간소화

## 반영 내용
- `상담사근태_증빙반영` 시트를 제거했습니다. 기존 `상담사근태`가 증빙 반영 기준 원장입니다.
- `관리자수정 충돌` 시트를 제거했습니다. 인력DB의 사번별 담당 매니저를 기준으로 본인 상담사 수정본만 반영합니다.
- `출근증빙·휴무확인` 시트는 `상담사근태_관리자반영` 기준으로 생성합니다.
- 관리자 수정본에서 이미 휴무/연차/반차/경조/공가/출산휴가/육아휴직/대체/보상/출근시간 등으로 정리된 날짜는 출근증빙·휴무확인 대상에서 제외합니다.
- `관리자수정 비교` 시트는 유지하여 기존 `상담사근태`와 `상담사근태_관리자반영`의 차이만 확인할 수 있게 했습니다.

## 유지 기능
- v61 중간 저장 이력 선택/삭제, 전체 재분석 유지
- v62 매니저별 월마감 수정본 업로드 유지
- 증빙 반영 로직은 `상담사근태`와 `상담사근태_관리자반영` 모두에 유지
