# V41 주 근태 확인자 시트 추가

## 적용 내용
- 최종본 엑셀에 `주 근태 확인자` 시트 추가
- 월요일~일요일 기준 ISO 주차 계산
- 2026년 6월: W23~W27, 7월: W27~W31, 8월: W31~W36 자동 표시
- 실제 출근기록이 주 6일 이상 또는 완성된 주차에서 주 3일 이하인 직원 표시
- 월 경계 주차는 직전 월 D1 월 마감의 `daily_statuses_json`과 현재 월 자료를 합산
- 끝나지 않은 주차는 `누적 중`으로 표시하고 3회 이하 판정을 보류
- 이상 주차 칸에 체크 및 출근 횟수 표시
- 비고에 `W주차(기간) 월~일 상태` 형식으로 7일 계획·근태 표시
- 서울/경인/충청/경북/경남/전라 지역별 그룹
- 네이비 대시보드, 7행 고정, 70% 배율, 눈금선 숨김

## 저장 기준
- 신규 D1 테이블이나 마이그레이션 없음
- 기존 월 마감의 일별 스냅샷을 사용
- 월 경계 누적을 위해 직전 월 마감 저장이 필요
