# V48 중간 확인 저장 UI 명확화

- 기존 v47에는 중간 확인 완료 파일을 별도로 넣는 전용 명칭이 없어 사용자가 기록 위치를 알기 어려웠음.
- 분석 화면의 선택 파일명을 `증빙·중간 확인 완료 파일`로 변경.
- 7/10처럼 월중 기준일까지 확인한 파일은 해당 칸에 넣고 분석 후 `중간 확인 저장`을 누르도록 안내 문구 추가.
- 기준일이 월말 이전이면 저장 버튼이 `중간 확인 저장`으로, 월말이면 `월 마감 교체 저장`으로 자동 변경.
- 기존 D1 구조와 저장 API를 그대로 사용하므로 마이그레이션 없음.
- 같은 월 중간 확인 저장 후 다음 분석 시 기존 `currentMonthDailyFacts` 이어받기 로직을 사용.
