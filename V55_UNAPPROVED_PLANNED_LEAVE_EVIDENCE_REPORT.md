# V55 미승인 계획 연차 증빙 반영 수정

## 수정 목적
- 근무계획에 연차/반차/휴가성 코드가 있어도 연차신청현황에서 승인/완료가 아니면 정상 처리하지 않도록 수정
- 승인되지 않은 계획 연차를 상담사근태에서 경고색으로 표시
- 승인되지 않은 계획 연차를 출근증빙·휴무확인 시트에 자동 노출

## 주요 변경
1. 연차신청현황 전체 상태 맵 추가
   - 승인/완료
   - 미승인
   - 반려/취소/철회
   - 신청없음

2. 상담사근태 반영 기준
   - 승인/완료 + 출근 없음: 연차/반차/휴가 색상 정상 적용
   - 미승인/반려/신청없음 + 출근 없음: 계획값은 표시하되 경고색 적용
   - 실제 출근기록/출근증빙 있음: 기존처럼 출근시간 우선

3. 출근증빙·휴무확인 시트
   - 계획 연차/반차/출산휴가/육아휴직/공가/경조/대체/보상인데 승인 건이 없으면 자동 노출
   - 미승인/반려/신청없음 사유가 상세사유에 표시
