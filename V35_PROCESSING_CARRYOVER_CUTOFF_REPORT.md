# v35 처리색상·이월잔여·기준일 수정 보고서

## 1. 계획&근태 상이 처리 완료 색상

- M열에 `O`, `○`, `ㅇ` 입력 시 N열 수식이 `처리 완료`로 변경됩니다.
- `workbook.xml`에서 `계획&근태 상이 인원`이 `계획&amp;근태 상이 인원`으로 저장되어 시트를 찾지 못하던 문제를 수정했습니다.
- XLSX 내부 조건부서식 DXF를 표준 OOXML 방식인 `patternType="solid" + fgColor`로 저장하도록 수정했습니다.
- 따라서 파일을 다시 업로드하기 전에도 Excel 화면에서 M열과 N열이 즉시 연녹색으로 변경됩니다.
- 기존의 `bgColor` 단독 방식으로 인해 검은색 또는 색상 미변경으로 보이던 문제를 제거했습니다.

## 2. 구분 색상 안내

### 계획&근태 상이 인원
- 근무인데 출근기록 없음: 빨강
- 휴무·휴가인데 출근기록 있음: 주황
- 검토 필요: 파랑
- 처리 완료: 초록

### 휴무 초과자
- 이월 대체·보상 잔여: 파랑
- 휴무초과 확인 요청: 빨강
- 휴무초과 대체 완료: 초록
- 처리 완료: 초록

병합된 안내 영역 전체에 채우기와 글자색을 직접 적용하여 검은색으로만 보이지 않게 했습니다.

## 3. 휴무 초과자 이월 잔여

직원별로 다음 열을 추가했습니다.

- 이월 대체잔여
- 이월 보상잔여
- 이월 합계

이월값은 대상 월 시작 직전 월말까지 사용한 뒤 남은 lot 중, 대상 월에 사용할 수 있는 대체휴무·보상휴가를 합산합니다. 사용 시작일이 전월 이전이고 대상 월까지 유효하면 미래 발생월로 등록된 부여분도 기존 운영 기준대로 포함됩니다.

새 열 구성에 따라 처리 입력은 T열, 처리상태는 U열입니다.

## 4. 기준일 이후 제외

- 기준일이 `2026-06-28`이면 비교 최종본에 존재하는 `2026-06-29`, `2026-06-30` 값은 비교 차이로 생성하지 않습니다.
- 전체 요약본에서도 기준일 이후 비교 건을 제외합니다.

## 5. 처리 완료 건 전체 요약 제외

- 계획&근태 상이 시트에서 O 처리된 `사번+발생일`은 재분석 시 `resolved`로 반영합니다.
- 해당 키는 전체 요약본과 비교 최종본 차이 생성에서 모두 제외합니다.
- 휴무 초과자 O 처리도 기존과 같이 전체 요약본에서 제외합니다.
