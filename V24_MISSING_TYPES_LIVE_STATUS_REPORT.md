# v24 수정 내역

## 1. 상담사근태 미입력 구분

- `출근 미입력`: 근무계획은 있으나 실제·변경 출근시간이 없음
- `계획 미입력`: 출근기록은 있으나 해당 날짜 근무계획이 없음
- `출ㆍ계 미입력`: 근무계획과 출근기록이 모두 없음
- 세 문구는 8pt로 표시하며 빨강·노랑·자주 계열로 구분

## 2. 출근 미등록 시트

- J열 구분 배지를 미입력 유형별로 다르게 표시
- 지역 요약행도 유형별 건수를 각각 집계
- K열 O/○/ㅇ 입력 시 상단 카드와 지역별 완료 건수 갱신
- 처리상태 M열이 `미처리`에서 `처리 완료`로 변경될 때 연녹색으로 즉시 변경
- K열 입력칸도 완료 시 연녹색으로 변경

## 3. 증빙 연동 수정

- 상담사근태 수식이 기존 잘못된 I열 날짜 참조가 아니라 `출근 미등록 H열 발생일`을 참조하도록 수정
- K열 O 입력 시 상담사근태 해당 날짜가 `출근`으로 계산되고 `#A9D08E` 조건부서식 적용
- 재분석 시 단순 출근 미입력은 이상 목록에서 제거
- 출ㆍ계 미입력은 출근 누락만 해소하고 계획 미입력 검토는 유지
- 휴무·연차 계획의 증빙 출근은 계획&근태 상이에 유지

## 4. 유지 사항

- D1 테이블·기존 데이터 삭제 없음
- 같은 경로·같은 월 재등록 시 기존 월 자료 교체 방식 유지
- 발생일 대체휴무 계산 로직 유지
