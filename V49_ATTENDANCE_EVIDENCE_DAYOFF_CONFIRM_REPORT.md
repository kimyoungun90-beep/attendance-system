# V49 출근증빙·휴무확인 시트 반영 보고

## 반영 내용
- `상담사근태` 바로 다음 2번째 시트로 `출근증빙·휴무확인` 시트를 배치했습니다.
- 실제 출근시간이 이미 있는 직원/일자는 확인 대상에서 제외됩니다.
- 계획이 휴무인데 실제 출근시간이 있는 건은 출근으로 이미 인정되므로 확인 대상에서 제외됩니다.
- 실제 출근기록이 없는 건만 `출근증빙·휴무확인` 시트에 표시합니다.
- `휴무확인(O 입력)`에 O를 입력하면 재업로드 시 해당 직원·일자가 `휴무`로 확정됩니다.
- `출근확인(O 입력)`에 O를 입력하면 재업로드 시 해당 직원·일자가 `09:00` 출근으로 확정됩니다.
- 같은 엑셀 안에서도 `상담사근태` 날짜 셀이 `출근증빙·휴무확인` 시트의 K/L열을 참조해 즉시 표시가 바뀌도록 수식을 연결했습니다.
- 기존 `출근 미등록` 시트명은 `출근증빙·휴무확인`으로 대체했습니다.

## 사용 방법
1. 최종본 엑셀 다운로드
2. 2번째 시트 `출근증빙·휴무확인` 확인
3. 휴무가 맞으면 K열 `휴무확인(O 입력)`에 O
4. 출근 증빙이 맞으면 L열 `출근확인(O 입력)`에 O
5. 저장한 파일을 웹앱의 `증빙·중간 확인 완료 파일`에 다시 업로드
6. 다시 분석하면 상담사근태와 관련 집계가 확정값 기준으로 재계산됩니다.
