# v40 휴무·대체(보상) 조율 판정 수정

## 휴무 초과자 시트 신규 열
- 기본휴무
- 이월 대체(보상)잔여
- 당월 대체 발생
- 이월+당월 대체(보상) 합계
- 당월 휴무 사용 개수
- 휴무+대체(보상)사용 개수
- 휴무 초과 개수
- 대체 초과 개수
- 당월 사용 후 잔여 대체(보상)
- 판정
- 처리여부 / 처리상태

## 판정
- 기본휴무와 대체(보상)를 합친 전체 가능량보다 총 사용량이 많음: `휴무 초과 확인 요청` (빨강)
- 휴무만 초과했으나 남은 대체(보상)로 조정 가능: `휴무 N일 초과 · 대체(보상) 조율 필요` (주황)
- 대체(보상)만 초과했으나 남은 기본휴무로 조정 가능: `대체(보상) N일 초과 · 휴무 조율 필요` (주황)
- 별도 조율 불필요: `이상 없음` (초록)

## 계산 기준
- 당월 대체 발생 = 당월 전체 사용 가능 대체(보상) - 이월 잔여
- 휴무+대체(보상)사용 = 당월 휴무 사용 + 계획표에 실제 입력된 대체휴무·보상휴가 사용
- 당월 사용 후 잔여 = 이월+당월 발생 합계 - 직접 사용한 대체(보상) - 기본휴무 초과분
- 처리여부 입력 열은 R열, 처리상태는 S열
