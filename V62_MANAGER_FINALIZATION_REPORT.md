# V62 Manager Monthly Closing Corrections

## 목적
- 전자랜드 매니저별 월마감 수정본 엑셀을 웹앱에서 선택할 수 있게 추가했습니다.
- 기존 v61 기능은 유지하면서, 엑셀 저장 시 증빙 반영본과 관리자 반영본을 별도 시트로 생성합니다.

## 추가 UI
- 상단 메뉴: `월마감 수정본 반영`
- 전자랜드 매니저 14명별 파일 선택 카드
- 근태 종합 점검 화면에도 현재 선택된 수정본 개수를 표시

## 결과 엑셀 추가 시트
- `상담사근태_증빙반영`: 원본 + 증빙 + 승인 연차/휴가 기준
- `상담사근태_관리자반영`: 증빙반영본 + 매니저 월마감 수정본 날짜별 값
- `관리자수정 비교`: 두 시트가 다른 날짜만 표시
- `관리자수정 충돌`: 같은 사번·일자를 서로 다른 매니저 파일이 수정한 경우 자동 반영 제외

## 저장 처리
- 월마감 저장 시 선택한 매니저 수정본 원본 파일도 보관함에 함께 저장합니다.
- 월마감 저장 payload의 employeeFacts에는 비충돌 관리자 수정값을 dailyStatuses에 반영하도록 보정했습니다.

## 주의
- v62는 전자랜드 매니저 14명 기준으로 우선 적용했습니다.
- 홈플러스 매니저별 목록은 추후 명단 확정 후 같은 구조로 확장할 수 있습니다.
