# V37 연차 승인·색상·조건부서식 수정 보고서

## 연차 등록 현황 및 일자
- 신청상태가 `대기`여도 계획과 신청 종류가 같고 실제근태 조건이 맞으면 `동일`로 판정
- 하루 연차: 계획 연차 + 신청 연차 + 미출근 = 동일
- 오전/오후 반차: 계획 반차 + 신청 반차 + 출근기록 있음 = 동일
- 계획 연차/반차 + 신청 없음 = 빨간색 확인 요청
- 하루 연차 + 출근기록 있음 = 빨간색 출근기록 확인
- 반차 + 출근기록 없음 = 빨간색 출근기록 확인
- 계획과 신청 종류가 다름 = 주황색 계획·신청 다름
- O열 확인상태
  - 승인 상태: 승인 완료(하늘색)
  - 대기 상태: 승인 대기(노란색)
  - 오류/불일치: 확인 요청(빨간색)
  - 수기 `확인 완료`: 초록색
- 이름(G열)과 대조구분(M열)을 동일 색상 계열로 표시

## 검은색 채우기 오류
- 출근 미등록, 계획&근태 상이, 휴무 초과자, 매니저별 이상 근태의 O 입력/상태 조건부서식 수정
- Excel 호환 DXF 구조(`bgColor`)로 변경하여 검은색 채우기 현상 제거
- O 입력: 연녹색
- 처리/전달 완료: 연녹색
- 미처리/미전달: 연분홍색

## 매니저별 이상 근태
- 복사용 멘트에서 사번 제거, 이름만 표시
- 상단 구분 색상 안내와 행별 구분 색상 유지
