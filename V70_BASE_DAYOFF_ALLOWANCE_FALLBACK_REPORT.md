# V70 기본 휴무 기준 0 표시 수정

## 문제
상담사근태_관리자반영 일부 행에서 `휴무 가능 개수`가 0으로 표시되어, 전자랜드 6월 정상 휴무 8개가 전부 초과로 계산되는 문제가 있었습니다.

## 원인
일부 상담사 요약 데이터의 `baseAllowance`가 0 또는 누락된 상태로 전달되면 최종 엑셀 생성 단계에서 그대로 0을 사용했습니다.

## 수정
- 기본 휴무 기준 보정 함수 추가
- 전자랜드: 대상월 토·일 개수로 보정
- 홈플러스: 6개로 보정
- `상담사근태`, `상담사근태_관리자반영`, 자동 `휴무(공백)`, 휴무 등록/초과 계산 모두 같은 기준 사용
